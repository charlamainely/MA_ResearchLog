Week 2 - Designing the Whole Box

Dates: 2026-06-01 → 2026-06-07

Activities

  • Redesign board game and conversation deck
  • Develop story archive flow and site design
  • Refine digital portal and start overall packaging development

Reflection

This week I focused much more heavily on design tasks. First, I redesigned the board game and the card deck so that they feel more coherent as part of one boxed experience. Since I already decided last week that the prototypes should be played in sequence, I now had to think not only about each object individually, but also how they visually and structurally relate to one another. The board game especially needed a clearer and more refined layout after all the mechanical changes I had been making.

I also started developing the ideas for the archive of stories, which is the part of the project that would preserve and display the stories collected through the playful prototypes. I wanted this archive to still feel playful and approachable rather than like a plain database, so I worked on the flow and site design in a way that could keep the spirit of the project intact. This stage was still quite exploratory, but it helped me clarify how the digital archive might complement the physical kit rather than just sit beside it.

In terms of digital development, after refining the board game mechanics last week, I also started refining the digital portal so that it matches the new version of the game. The portal was still using content and logic from the older iteration, so a lot of this work was really about updating the system so it does not fall out of sync with the new board and new mechanics. This is honestly one of the trickier parts of having a hybrid game, because every change to the physical design tends to require some change to the digital system as well.

I also started looking into the overall packaging of the game kit, including different formats, sizes, paper quality, and how the whole set might physically come together. At this stage I was working on the box design in English first, with the plan to translate everything into Vietnamese later for user testing. Even just thinking through the size and format of the box already made me realise how many practical decisions I would need to make to turn the project into something that actually feels like a finished product.
