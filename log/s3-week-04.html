Week 4 - Back to Vietnam for Testing

Dates: 2026-06-15 → 2026-06-21

Activities

  • Travel back to Vietnam for user testing and print production
  • Translate all game kit content into Vietnamese and test print materials
  • Conduct first user testing session and present research virtually at SEAPAVAA

Reflection

This week I travelled back to Vietnam with the goal of conducting user testing, as well as printing out my packaging and game content. Because so much of this project depends on family-based testing in a Vietnamese context, it felt necessary to be back home and handle the printing and facilitation more directly. There was a lot to prepare all at once, but it also felt like the project was finally moving into a much more real and embodied stage.

However, I soon discovered that the box design I had originally intended would be not only impractical, but also very expensive to produce. So I had to adapt the design into a different, more budget-friendly format. This was a little frustrating, because I had already been attached to the earlier idea, but realistically I need the final package to be something I can actually afford to print and reproduce for testing.

I also translated all the content into Vietnamese this week, including the activity book, the cards, the board game, and the box packaging, and did a couple of test prints. This was honestly one of the more painful parts of the process, because a lot of concepts that make sense in English do not always have a clean or elegant translation in Vietnamese. On top of that, I am very used to writing in English, and there are many terms that people would casually use in English even in Vietnamese-speaking contexts. But I really want this whole kit to be fully accessible in Vietnamese, so that even players who do not speak a word of English can still understand and participate in the experience.

I also tried developing a multiplayer platform that would allow players to join the intergenerational board game from different devices. Unfortunately, it was not ready by the time I needed to conduct the user testing, so I had to fall back on the backup version I developed a few weeks ago, where all players join using one shared device. This was not ideal, but I am glad I had the backup in place, otherwise I would have been completely stuck.

I had my first user testing session this week, and the results from that session are documented separately in the fieldwork section. Even though I still need to synthesise the findings more properly later, it already gave me useful information on how the new iteration is working in practice and what still needs to change before the next tests.

Since I was already back in Vietnam, I also presented my research virtually at the SEAPAVAA conference this week. It was nice to be able to share the project in a more formal research context while I was actively in the middle of testing it. I received a few good questions about how this research might be adapted into different cultural and national contexts while still maintaining a level of sensitivity to local family structures, values, and communication norms. I found these questions really useful, because although my project is currently grounded in Vietnamese families, I do think there is future potential for the framework to be adapted elsewhere. At the same time, the discussion reminded me that any such adaptation would need to be done very carefully, and not by assuming that the same prompts, game structures, or family dynamics can simply be transplanted across contexts without change.

This is something I will definitely take into consideration for future development, especially if the project moves beyond the Vietnamese context later on. For now, it was encouraging to know that the research could spark this kind of discussion, even while it is still being actively tested and refined.
